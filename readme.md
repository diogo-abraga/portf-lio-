🚀 Meu Portfólio Profissional | Diogo Braga
Este é o repositório do meu portfólio pessoal, desenvolvido para apresentar minha trajetória, projetos e habilidades técnicas como Desenvolvedor Front-end. O site foi construído com foco em performance, responsividade e uma narrativa visual que conecta minha experiência no varejo com o desenvolvimento de software.

🔗 Demonstração
Acesse o site ao vivo: [(https://diogo-abraga.github.io/portf-lio-/)](https://diogo-abraga.github.io/portf-lio-/)

💻 Sobre o Projeto
O projeto nasceu da necessidade de criar uma identidade única para a minha transição de carreira. Saindo do setor de vendas e promoção para a tecnologia, utilizei este site para aplicar conceitos avançados de manipulação de DOM, estilização moderna e componentes reutilizáveis.

Principais Seções:
Hero Section: Apresentação de impacto com foco em conversão.

Stacks: Vitrine das tecnologias dominadas (React, JS, CSS3).

Galeria de Trajetória: Storytelling visual da minha evolução profissional.

Diferenciais: Foco em soft skills como Solução de Problemas e Visão de Mercado.

Contato: Integração direta com redes profissionais (LinkedIn, GitHub, WhatsApp).

🛠️ Tecnologias Utilizadas
HTML5 Semântico: Para melhor SEO e acessibilidade.

CSS3 Custom Properties: Uso de variáveis para manter a paleta Roxo & Dourado consistente.

JavaScript (ES6+): Lógica para seções interativas e navegação.

Font Awesome: Para iconografia técnica e profissional.

Responsividade: Design Mobile-First adaptável a qualquer dispositivo.

📈 Diferenciais Técnicos
Este portfólio reflete meu compromisso com a qualidade:

Clean Code: Estrutura de arquivos organizada e fácil manutenção.

Performance: Otimização de imagens e scripts para carregamento rápido.

UI/UX: Interface pensada para facilitar a leitura dos recrutadores.
